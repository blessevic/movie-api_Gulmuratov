# Movie API 

RESTful API для управления фильмотекой. Позволяет управлять фильмами, жанрами и отзывами с поддержкой массовых операций и JWT-аутентификации.

## Возможности

-  Получение списка фильмов с фильтрацией, поиском и сортировкой
-  Создание одного или нескольких фильмов за раз
-  Обновление одного или нескольких фильмов (полное и частичное)
-  Удаление одного или нескольких фильмов
-  Управление жанрами и отзывами
-  Анонимное создание отзывов (автор не требуется)
-  JWT-аутентификация для защищённых операций
-  Автоматическая документация Swagger UI
-  Кеширование GET-запросов (15 минут)

##  Эндпоинты API

**Базовый URL:** `http://127.0.0.1:8000/api/v1/`

### 1. Жанры (`/genres/`)

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/genres/` | Список всех жанров |
| GET | `/genres/?search=драма` | Поиск по названию |
| GET | `/genres/{id}/` | Получить один жанр |
| POST | `/genres/` | Создать один жанр |
| POST | `/genres/` | Создать несколько жанров (массив) |
| PUT | `/genres/{id}/` | Полное обновление жанра |
| PATCH | `/genres/{id}/` | Частичное обновление жанра |
| PATCH | `/genres/` | Массовое частичное обновление (массив с id) |
| DELETE | `/genres/{id}/` | Удалить один жанр |
| DELETE | `/genres/batch/?ids=1,2,3` | Массовое удаление |

### 2. Фильмы (`/movies/`)

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/movies/` | Список всех фильмов |
| GET | `/movies/?release_year=1994` | Фильтрация по году выпуска |
| GET | `/movies/?rating__gte=8.5` | Фильтрация по рейтингу (>=) |
| GET | `/movies/?search=побег` | Поиск по названию/описанию |
| GET | `/movies/?ordering=-rating` | Сортировка по убыванию рейтинга |
| GET | `/movies/{id}/` | Получить один фильм |
| POST | `/movies/` | Создать один фильм |
| POST | `/movies/` | Создать несколько фильмов (массив) |
| PUT | `/movies/{id}/` | Полное обновление фильма |
| PATCH | `/movies/{id}/` | Частичное обновление одного фильма |
| PATCH | `/movies/` | Массовое частичное обновление (массив с id) |
| DELETE | `/movies/{id}/` | Удалить один фильм |
| DELETE | `/movies/batch/?ids=1,2,3` | Массовое удаление |

### 3. Отзывы (`/reviews/`)

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/reviews/` | Список всех отзывов |
| GET | `/reviews/?movie_id=1` | Фильтрация по фильму |
| GET | `/reviews/?rating=5` | Фильтрация по оценке |
| GET | `/reviews/{id}/` | Получить один отзыв |
| POST | `/reviews/` | Создать один отзыв |
| POST | `/reviews/` | Создать несколько отзывов (массив) |
| PUT | `/reviews/{id}/` | Полное обновление отзыва |
| PATCH | `/reviews/{id}/` | Частичное обновление одного отзыва |
| PATCH | `/reviews/` | Массовое частичное обновление |
| DELETE | `/reviews/{id}/` | Удалить один отзыв |
| DELETE | `/reviews/batch/?ids=1,2,3` | Массовое удаление |

> **Примечание по отзывам:** создавать могут все (в т.ч. анонимы), изменять и удалять — только авторизованные пользователи.

## 🔐 Аутентификация

API использует **JWT-токены** (`djangorestframework-simplejwt`).

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST | `/api/token/` | Получить access и refresh токены |
| POST | `/api/token/refresh/` | Обновить access токен |

**Добавление токена в запрос:**
`Authorization: Bearer <ваш_access_токен>`

## Установка и запуск


# 1. Клонировать репозиторий
git clone https:https://github.com/blessevic/movie-api_Gulmuratov

# 2. Создать и активировать виртуальное окружение
python -m venv venv

# venv\Scripts\activate       # Windows

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Выполнить миграции
python manage.py migrate

# 5. Создать суперпользователя (для админки и генерации токенов)
python manage.py createsuperuser

# 6. Запустить сервер
python manage.py runserver
