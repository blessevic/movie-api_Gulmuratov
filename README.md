 Task Manager API

API для управления задачами, категориями и комментариями.  
Позволяет:

- Управлять категориями задач
- Создавать, отслеживать и обновлять задачи
- Оставлять комментарии к задачам (с поддержкой анонимов и авторизованных пользователей)
- Выполнять массовые операции (создание, обновление, удаление)

---

## Эндпоинты API

**Базовый URL:** `http://127.0.0.1:8000/api/v1/`

### 1. Категории (`/categories/`)

| Метод   | Эндпоинт                         | Описание                                      |
|---------|----------------------------------|-----------------------------------------------|
| GET     | `/categories/`                   | Список всех категорий                         |
| GET     | `/categories/?name=работа`       | Фильтрация по названию (частичное совпадение) |
| GET     | `/categories/{id}/`              | Получить одну категорию                       |
| POST    | `/categories/`                   | Создать одну категорию                        |
| POST    | `/categories/`                   | Создать несколько категорий (массив)          |
| PUT     | `/categories/{id}/`              | Полное обновление категории                   |
| PATCH   | `/categories/{id}/`              | Частичное обновление одной категории          |
| PATCH   | `/categories/`                   | Массовое обновление (массив с `id`)           |
| DELETE  | `/categories/{id}/`              | Удалить одну категорию                        |
| DELETE  | `/categories/batch/?ids=1,2,3`   | Массовое удаление                             |

### 2. Задачи (`/tasks/`)

| Метод   | Эндпоинт                                   | Описание                                      |
|---------|--------------------------------------------|-----------------------------------------------|
| GET     | `/tasks/`                                  | Список всех задач                             |
| GET     | `/tasks/?category_id=1`                    | Фильтрация по категории                       |
| GET     | `/tasks/?status=new`                       | Фильтрация по статусу (`new`, `in_progress`, `done`) |
| GET     | `/tasks/{id}/`                             | Получить одну задачу                          |
| POST    | `/tasks/`                                  | Создать одну задачу                           |
| POST    | `/tasks/`                                  | Создать несколько задач (массив)              |
| PUT     | `/tasks/{id}/`                             | Полное обновление задачи                      |
| PATCH   | `/tasks/{id}/`                             | Частичное обновление одной задачи             |
| PATCH   | `/tasks/`                                  | Массовое обновление (массив с `id`)           |
| DELETE  | `/tasks/{id}/`                             | Удалить одну задачу                           |
| DELETE  | `/tasks/batch/?ids=1,2,3`                  | Массовое удаление                             |

### 3. Комментарии (`/comments/`)

| Метод   | Эндпоинт                                   | Описание                                      |
|---------|--------------------------------------------|-----------------------------------------------|
| GET     | `/comments/`                               | Список всех комментариев                      |
| GET     | `/comments/?task_id=1`                     | Фильтрация по задаче                          |
| GET     | `/comments/{id}/`                          | Получить один комментарий                     |
| POST    | `/comments/`                               | Создать один комментарий (анонимно или с авторизацией) |
| POST    | `/comments/`                               | Создать несколько комментариев (массив)       |
| PUT     | `/comments/{id}/`                          | Полное обновление комментария (только для авторизованных) |
| PATCH   | `/comments/{id}/`                          | Частичное обновление одного комментария       |
| PATCH   | `/comments/`                               | Массовое обновление                           |
| DELETE  | `/comments/{id}/`                          | Удалить один комментарий (только для авторизованных) |
| DELETE  | `/comments/batch/?ids=1,2,3`               | Массовое удаление                             |

> **Примечание по комментариям:**  
> - Создавать могут все (в т.ч. анонимы).  
> - Изменять и удалять – только авторизованные пользователи.

## Аутентификация

API использует **JWT-токены** (библиотека `djangorestframework-simplejwt`).

| Метод | Эндпоинт                | Описание                         |
|-------|-------------------------|----------------------------------|
| POST  | `/api/token/`           | Получить `access` и `refresh` токены (передать `username`/`password`) |
| POST  | `/api/token/refresh/`   | Обновить `access` токен          |

После получения токена добавьте его в заголовок каждого запроса:  
`Authorization: Bearer <ваш_access_токен>`

## Установка и запуск


# 1. Клонировать репозиторий
git clone https://github.com/theterorrplaygt/practice6.git

# 2. Создать и активировать виртуальное окружение
python -m venv venv

# venv\Scripts\activate       # Windows

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Выполнить миграции
python manage.py migrate

# 5. Создать суперпользователя (для админки и генерации токенов)
python manage.py createsuperuser

# 6. Запустить сервер
python manage.py runserver
